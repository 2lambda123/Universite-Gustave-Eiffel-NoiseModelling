Scripts de conversion de fichiers XLS contenant le traffic.

Nécessite le module pyExcelerator :
http://sourceforge.net/projects/pyexcelerator/

Le module dbfUtils a été ajouté car il n'a pas de système de suivi de version.

How to run :

python script_convert_troncons.py "troncons_calage_2008_v5.xls" "Services TC 2008.xls" "Tps tc troncons.xls"
